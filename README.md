# Yenoo Labs 🧪

**Official landing page for Yenoo Labs and Spendly — a private, ad-free personal finance tracker for Android.**

This repository hosts the website via GitHub Pages. It serves as the main information hub for our applications and provides a safe, Google Play-compliant portal for users who wish to support the development.

🔗 **[Visit the live website here](https://yenoo7.github.io/yenoo/)**

---

## 📱 Featured App: Spendly

Spendly was built natively for Android using **Kotlin and Jetpack Compose**. It was generated entirely from a detailed Product Requirements Document (PRD) using the **OpenCode AI agent**—achieving the core architecture, UI, and logic with zero manual coding.

### Key Features:
- **Multi-Currency Support:** 50+ currencies with automatic exchange rate updates.
- **Privacy First:** 100% ad-free, no user tracking, and optional Biometric locks.
- **Cloud Backups:** Encrypted, PIN-protected backups straight to Google Drive.
- **Data Management:** Easy Excel import/export functionality.

📲 **[Download Spendly on Google Play](https://play.google.com/store/apps/details?id=com.spendlywallet.expensetracker)**
